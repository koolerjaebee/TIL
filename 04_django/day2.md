## html

### tag

- id : html 상에서만 구별하는 이름
- name : 서버로 보내는 데이터의 변수 이름
- value : 서버로 보내는 데이터의 값
- for : html 상에서 무엇을 봐야하는지 알려줘야함 (id값 입력)
- action : form 태그에서 어디로 보내야할지 정함 (URL)
