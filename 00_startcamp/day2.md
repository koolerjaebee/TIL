# what is git

git: scm(source code manager) / vcs(version control system)

git ==> version 관리를 해주는 감시카메라

`git add '<filename>'` add == 파일 commit 준비 aka "찍습니다"

`git commit -m '<message>'` commit == 상태 저장 aka ''찰칵''

